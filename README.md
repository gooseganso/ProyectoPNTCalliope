## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://makecode.calliope.cc/](https://makecode.calliope.cc/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/gooseganso/proyecto** and import

## Edit this project ![Build status badge](https://github.com/gooseganso/proyecto/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* open [https://makecode.calliope.cc/](https://makecode.calliope.cc/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/gooseganso/proyecto** and click import
